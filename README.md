git-trello
==========

Custom git command for tagging commits with trello cards.

Installation
------------
Add `bin/` to your `PATH`. Eg.

    git clone git@github.com:saifelse/git-trello.git ~/git-trello
    echo 'export PATH=~/git-trello/bin:$PATH' >> ~/.zshrc
    source ~/.zshrc
